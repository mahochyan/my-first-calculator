# Instructions for AI Agents

This repository is a progressive Python learning path for beginners. When working with this code, please keep the following in mind:

## Environment Requirements
- **Python 3** is required to run all scripts in this repository.

## Script Execution
- Several scripts in the `tutorial/` directory are interactive and use the `input()` function:
    - `tutorial/01_basics/basics.py`
    - `tutorial/02_control_flow/control_flow.py`
- When running these scripts in an automated or non-interactive environment, you must provide piped input (e.g., `echo "input_value" | python3 script.py`).

## Language
- Documentation, tutorials, and code comments are primarily in **Chinese**.

## Project Structure
- `tutorial/`: Contains 4 levels of learning materials.
- `GITHUB_SETUP_GUIDE.md`: Contains instructions for Git/GitHub configuration.
