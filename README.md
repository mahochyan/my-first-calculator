# 🚀 Python 初学者学习之路 (Python for Beginners)

欢迎来到 Python 的世界！这个仓库已经被重新组织，为你提供一个完整的、分阶段的 Python 学习路径。

## 📁 学习路径结构

我们为你准备了 4 个渐进式的学习阶段：

### 1. [基础入门 (Basics)](./tutorial/01_basics/)
*   **学习重点**: 变量、数据类型、输入和输出。
*   **核心文件**: `basics.py`

### 2. [控制流 (Control Flow)](./tutorial/02_control_flow/)
*   **学习重点**: 条件判断 (`if/else`) 和循环 (`for/while`)。
*   **核心文件**: `control_flow.py`

### 3. [数据结构 (Data Structures)](./tutorial/03_data_structures/)
*   **学习重点**: 列表 (List) 和字典 (Dictionary)。
*   **核心文件**: `data_structures.py`

### 4. [函数与实战 (Functions & Project)](./tutorial/04_functions/)
*   **学习重点**: 如何编写和调用函数。
*   **核心文件**: `calculator.py` (你的第一个计算器小项目！)

---

## 🛠️ 如何开始？

1.  **从 Level 01 开始**: 进入 `./tutorial/01_basics/` 目录，阅读 `README.md`，然后运行 `basics.py`。
2.  **动手尝试**: 尝试修改每一段代码，看看会有什么变化。
3.  **循序渐进**: 按照数字顺序完成每一关的学习。
